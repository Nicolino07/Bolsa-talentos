Flujo del programa/datos

SQL → Python → Prolog → Python → API → Front

Ejemplo real de flujo eficiente

1 - Python pregunta a SQL:

SELECT p.dni, a.id_actividad, a.nivel
FROM persona p
JOIN persona_actividad a ON p.dni = a.dni;


2 - Convierte resultados en hechos:

persona_actividad(DNI, ID_Actividad, Nivel).


3 - Prolog aplica reglas:

match(Persona, Oferta) :- 
    requiere_actividad(Oferta, X),
    tiene_actividad(Persona, X).




➡ Super eficiente y multiparadigma real
